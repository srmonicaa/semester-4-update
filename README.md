# Simulasi Ujian Mobile STSI4204 & STSI4207

## Publikasi dengan GitHub Pages
1. Buat repository baru di GitHub.
2. Upload file `index.html` ini ke halaman utama repository (root), bukan ke dalam folder dan bukan file ZIP.
3. Buka **Settings** > **Pages**.
4. Pada **Build and deployment**, pilih **Deploy from a branch**.
5. Pilih branch `main` dan folder `/(root)`, lalu tekan **Save**.
6. Setelah selesai, buka tautan GitHub Pages yang ditampilkan.

Aplikasi berjalan tanpa backend dan dapat dibuka di HP.
